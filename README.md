
# GPT-2 Web Chat (Node.js)

Small, local model chat using transformers.js (distilgpt2).
- Dark mode UI
- Chat history (localStorage)
- Math + daily Q&A (best-effort, small model)

## Run
```bash
npm install
npm start
```
Open http://localhost:3000

## Replit
- Create Node.js Repl
- Upload files or `git clone`
- Run `npm install` then `npm start`
